Cách lấy hình ảnh như trên:
b1: Tạo các file hình ảnh với kích thước khác nhau bằng công cụ online hoặc phần mềm chỉnh sửa ảnh.
b2: nhấn make favicons hoặc vào trang https://www.favicon-generator.org/
b3: tải lên hình ảnh chính (kích thước tối thiểu 260x260px)
b4: tải file .zip về và giải nén vào thư mục dự án
b5: chèn đoạn code trên vào thẻ của file html
